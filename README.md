# Vanilla JavaScript Todo App

A feature-complete Todo application built using **vanilla JavaScript**, focusing on clean state management, UI-driven logic, and persistent storage.

This project was built to strengthen core JavaScript fundamentals and mirror React-style state and rendering patterns without using any frameworks.
